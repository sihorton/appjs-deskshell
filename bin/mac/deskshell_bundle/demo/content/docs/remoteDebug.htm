Debug API

The deskShell.startApp function returns a promise, that is fullfilled with an app object. This app object has an rDebugApi object on it. The object implements the Remote Debugging Protocol (see https://developers.google.com/chrome-developer-tools/docs/protocol/1.1/index

To navigate to a new url you can do the following:

app.rDebugApi.pageNavigate('http://example.com/test.htm');
