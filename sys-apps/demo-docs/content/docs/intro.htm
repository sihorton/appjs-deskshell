Introduction

Deskshell can use an off the shelf chromium, a nodejs binary and create a desktop application from them. This means you can use the latest html5 technology but not be constricted by running inside the browser sandbox.

If you find missing information or things that are wrong then checkout the project and go to the sys-apps/default folder, find the html page and update it. Another alternative is to modify the project wiki.
