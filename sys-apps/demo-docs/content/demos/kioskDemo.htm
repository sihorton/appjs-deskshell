Kiosk Demo

By setting mode:kiosk in the application .desk file it will be launched fullscreen without any chrome. If the user does not have access to the keyboard then they cannot exit.

Note that currently there is some kind of chrome bug so if you launch this demo and have an already open chrome browser sometimes it launches within that instead of going full screen.
