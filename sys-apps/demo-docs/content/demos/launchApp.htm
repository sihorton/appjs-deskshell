launchApp

It is easy to launch another deskshell app, on the server side call:

var path = deskShell.appDir + "/../relative/path/to/app.desk";
deskShell.launchApp(f);

deskShell.appDir holds the directory that your application is running from.

You can test the functionality below, enter in a relative path to a .desk file and then click the button to start it.
