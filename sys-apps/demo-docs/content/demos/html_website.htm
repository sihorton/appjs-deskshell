Html Website

If you can create a simple html website then you can create a desk shell application.

create a folder called something like "htdocs" or "content" or similar. Copy all of the html, css, images, javascript and other files that make up your application. Then when you start the application give it the folder containing your html site

var running = deskShell.startApp({
	htdocs:htdocs
	,openSocket:true
	,launchChromium:true
	,exitOnChromiumClose:true
	,width:400
})
