Introduction

The DeskShell SDK lets you rapidly create desktop applications. You can easily edit the files on disk to alter how the application behaves and looks. When you then want friends or others to be able to use your application then you will need to package it up and deploy it.

Packaging takes all of the files in your application and then appends them into one large single file with an .appfs extension. You can then send that file to other computers and they can then run your application if they also have deskshell installed. The applications are generally portable which means that the same application can often run cross platform on windows, mac and linux.
